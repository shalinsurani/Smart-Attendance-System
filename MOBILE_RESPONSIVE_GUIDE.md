## ğŸ“ Common Responsive Classes

### Layout
```jsx
flex flex-col sm:flex-row          // Stack on mobile, row on desktop
grid grid-cols-1 md:grid-cols-2    // 1 col mobile, 2 cols desktop
hidden md:block                     // Hide on mobile, show on desktop
block md:hidden                     // Show on mobile, hide on desktop
```

### Spacing
```jsx
p-4 sm:p-6 lg:p-8                  // Padding scales up
gap-3 sm:gap-4 lg:gap-6            // Gap scales up
space-y-4 sm:space-y-6             // Vertical spacing scales up
```

### Typography
```jsx
text-sm sm:text-base md:text-lg    // Text size scales up
text-2xl sm:text-3xl md:text-4xl   // Heading scales up
leading-tight sm:leading-normal    // Line height adjusts
```

### Sizing
```jsx
w-full sm:w-auto                   // Full width mobile, auto desktop
max-w-sm sm:max-w-md lg:max-w-lg   // Max width scales up
h-auto sm:h-64                     // Height adjusts
```
